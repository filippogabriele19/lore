💡 What are Virtual Edges?

Standard static analysis tools only see imports and call chains. LORE mines your Git history and co-commit patterns to identify Virtual Edges—files that co-evolve together across commits without explicit code references.

If a modification is made to a file with strong virtual coupling, LORE automatically alerts developers and indexes these implicit dependencies in its Knowledge Graph to prevent regressions.
